# vite-react-ts boilerplate

## Run Locally

Clone the project

```bash
  git clone https://github.com/MOONcitizenX/vite-react-ts.git
```

Go to the project directory

```bash
  cd vite-react-ts
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```

## Build

To build this project run

```bash
  npm run build
```
